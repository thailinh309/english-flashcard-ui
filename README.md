## Developer guide

Using `rekit-studio`, `node-v10.23.0`, `npm-6.14.8`,

For Rekit, 
```bash
npm install -g rekit-studio
```

### Start
Open UI on port 3100

```bash
yarn
yarn start
```

Open studio on port 3333
```bash
rekit-studio -p 3333
```
